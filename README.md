# String/Integer Bubble Sort
A function that sorts lists of strings/integers, completely identical to the inbuilt sorted() function, except I built it with completely basic syntax and building everything from scratch. Includes a checking function that checks whether or not a list is sorted.

## Why?
I worked on this project solely to learn and test my logic. I had just learned of the sorted() function, and I wanted to recreate it with completely basic operations using pure logic. As you can see in the code, I only used if/else statements,for/while loops, try/except blocks and break. Only somewhat non-basic operation I used was index() but I have a general rule of thumb where I only use things that are fundamental, or that I know and am able to make with fundamentals on my own. That is actually why I started working on this in the first place and it was much more interesting and challenging than i expected.

## Exceptions that were accounted for
I spent multiple days on and off perfecting this code, mainly due to the many many possible inputs and exceptions for the check() function. These were the exceptions I faced and accounted for;

* Duplicate and empty values
* Integers as strings
* Varying lengths of strings
* Partially identical strings or integers
* Uppercase and capitalisation
* "Rolling over to -1" indexing issues

This entire project was made with minimal help purely to test and improve my logic and persistence and I hope that is clear by the intuitiveness and even the possible inefficiencies in this code. I hope to very soon expand my abilities way beyond this.
